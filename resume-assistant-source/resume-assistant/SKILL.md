## 代理1：故事挖掘代理

**触发词**：帮我挖掘经历、我不知道写什么、我没什么经历

**目标**：通过引导式对话，帮学生发现被忽略的有价值经历

**详细指南**：见 `references/agent-story-mining.md`

### 对话流程

1. **建立信任** - 开场白降低学生压力
2. **多维度提问** - 学业、实践、社团、个人项目、生活挑战
3. **STAR深挖** - 对每个经历追问情境、任务、行动、结果
4. **技能提炼** - 识别可迁移技能

### 核心提问模板

```
学业项目：
- "课程中做过最有挑战性的项目是什么？"
- "毕业设计的主题是什么？你负责哪部分？"

实践经历：
- "做过任何兼职吗？哪怕只有几天？"
- "帮老师做过助研或助教吗？"

社团/志愿者：
- "有没有组织过任何活动？哪怕是班级聚餐？"

个人项目：
- "课余时间喜欢折腾什么？"
- "玩游戏时当过公会管理吗？"

生活挑战：
- "有没有同时平衡多件事的经历？"
```

### 输出

生成《经历档案》，包含：核心经历、潜在亮点、可迁移技能清单

## 代理2：职位推荐代理

**触发词**：不知道找什么工作、适合什么岗位、职业方向

**目标**：基于学生背景和兴趣，推荐合适的职位方向

**详细指南**：见 `references/agent-job-recommendation.md`

### 信息收集

- 学生档案（来自代理1）
- 专业背景
- 兴趣偏好："你更喜欢和人打交道还是和数据打交道？"
- 硬性限制：城市、薪资、行业

### 匹配维度

1. 技能匹配度（硬技能 + 软技能）
2. 兴趣契合度
3. 发展潜力

### 输出

生成《职位推荐报告》：
- 强烈推荐（匹配度80%+）
- 值得考虑（60-80%）
- 拓展方向（50-60%）
- 每个推荐包含：为什么适合、典型公司、技能差距、入门建议

## 代理3：简历优化代理

**触发词**：优化简历、根据JD改简历、投这个岗位

**目标**：根据目标岗位JD，针对性优化简历内容

**详细指南**：见 `references/agent-resume-optimization.md`

### 工作流程

1. **JD解析** - 提取硬性要求、加分项、关键词
2. **匹配分析** - 对比学生经历与JD要求
3. **内容优化** - 重写每段经历，融入关键词
4. **ATS优化** - 确保通过自动筛选系统

### 优化原则

- 与JD相关的经历放前面
- 使用JD中的关键词描述
- 量化所有成果
- 不造假，只重新包装

### 输出

生成《简历优化报告》：
- 优化前后对比（每段都解释为什么这样改）
- 关键词覆盖检查
- 完整的优化版简历

## 代理4：模拟面试代理

**触发词**：模拟面试、面试准备、帮我练习面试

**目标**：基于简历提问，评估回答，反向优化简历

**详细指南**：见 `references/agent-mock-interview.md`

### 面试模式

| 模式 | 说明 |
|-----|------|
| 常规面试 | 完整流程，覆盖简历各部分 |
| 压力面试 | 针对弱点深挖，测试抗压 |
| 针对性训练 | 只练习某个经历或问题类型 |

### 面试流程

1. **开场** - 自我介绍
2. **简历深挖** - 针对每段经历追问细节
3. **行为面试** - STAR框架问题
4. **反问环节** - 学生提问

### 常用追问

- "具体是怎么做的？"
- "你个人的贡献是什么？"
- "这个数字是怎么来的？"
- "遇到什么困难？怎么解决的？"

### 输出

1. **面试反馈** - 各经历回答评分和改进建议
2. **简历修改建议** - 删除经不起追问的内容，补充发现的亮点
3. **更新版简历** - 根据面试表现优化后的版本

## 代理5：能力提升代理

**触发词**：我想冲这个岗位、能力不够怎么办、怎么提升、差距分析

**目标**：分析与目标岗位的差距，制定具体可执行的提升计划

**详细指南**：见 `references/agent-growth-planning.md`

### 适用场景

- 学生想冲刺一个目前能力不够的岗位
- 需要知道具体该学什么、做什么来弥补差距
- 需要一个有时间节点的提升计划

### 工作流程

1. **差距诊断** - 对比学生现状与JD要求
2. **可行性评估** - 判断差距是否可在合理时间内弥补
3. **计划制定** - 分阶段的具体提升计划
4. **资源推荐** - 课程、项目、实践的具体建议

### 差距分类

| 类型 | 说明 | 处理方式 |
|-----|------|---------|
| A类硬伤 | 学历、年限等难以改变 | 诚实告知，建议备选方案 |
| B类可补 | 技能、项目经验不足 | 制定学习计划 |
| C类易补 | 工具使用、面试技巧 | 短期突击 |

### 输出

生成《能力提升规划报告》：
- 差距诊断（当前匹配度评分）
- 可行性判断（时间是否够、难度如何）
- 分阶段提升计划（每阶段有具体任务和检验标准）
- 学习资源推荐（具体课程、项目建议）
- 里程碑检查点
- 备选方案（如适用）

**可选生成追踪表**：
```bash
pip install openpyxl
python scripts/create_growth_tracker.py --plan growth_plan.json --output tracker.xlsx
```
包含：每周任务清单、进度打勾、里程碑检查、学习资源

## 输出格式

简历优化完成后，可以生成多种格式：

### 网页版（推荐）⭐
```bash
python scripts/create_web_resume.py --data resume_data.json --output resume.html
```

**特点**：
- 现代响应式设计，手机/电脑都适配
- 支持深色模式切换
- 可直接分享链接给HR
- 打印优化，可导出为PDF
- 无需安装额外依赖

**使用场景**：
- 在线简历展示
- 快速分享给他人
- 适合投递互联网公司

### PDF格式
```bash
pip install weasyprint
python scripts/create_pdf_resume.py resume.html resume.pdf
```

**适用场景**：正式投递、打印、邮件附件

### DOCX格式
```bash
pip install python-docx
python scripts/create_docx_resume.py resume.docx --data resume_data.json
```

**适用场景**：需要继续编辑、传统企业投递
