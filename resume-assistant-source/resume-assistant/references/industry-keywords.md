# 行业关键词参考

## 技术/互联网行业

### 前端开发
- 技术栈：React, Vue, Angular, TypeScript, JavaScript, HTML5, CSS3, Webpack, Vite
- 能力词：响应式设计, 性能优化, 组件化开发, 状态管理, 跨端开发
- 成果词：页面加载速度, 首屏渲染, 用户体验, 代码复用率

### 后端开发
- 技术栈：Java, Python, Go, Node.js, Spring, Django, MySQL, Redis, MongoDB, Kafka
- 能力词：微服务架构, 分布式系统, 高并发处理, API设计, 数据库优化
- 成果词：QPS, 响应时间, 系统可用性, 并发量

### 移动开发
- 技术栈：iOS, Android, Swift, Kotlin, Flutter, React Native
- 能力词：原生开发, 跨平台开发, 性能调优, 内存管理
- 成果词：崩溃率, 启动时间, 用户留存, 应用商店评分

### 数据/AI
- 技术栈：Python, TensorFlow, PyTorch, Spark, Hadoop, SQL, Tableau
- 能力词：机器学习, 深度学习, 数据分析, 数据挖掘, 特征工程
- 成果词：模型准确率, 数据处理量, 预测精度, ROI提升

### DevOps/运维
- 技术栈：Docker, Kubernetes, Jenkins, Ansible, AWS, Azure, GCP
- 能力词：CI/CD, 容器化, 自动化部署, 监控告警, 故障排查
- 成果词：部署频率, 故障恢复时间, 系统稳定性, 成本优化

## 产品/运营

### 产品经理
- 工具：Axure, Figma, Jira, SQL, 数据分析工具
- 能力词：需求分析, 用户调研, 竞品分析, MVP, 迭代优化, 数据驱动
- 成果词：DAU/MAU, 转化率, 留存率, NPS, 用户增长

### 运营
- 工具：Excel, SQL, GA, 神策, GrowingIO
- 能力词：用户增长, 内容运营, 活动策划, 社群运营, 数据分析
- 成果词：GMV, 转化率, 复购率, 拉新成本, 用户LTV

## 金融行业

### 投资/分析
- 工具：Wind, Bloomberg, Python, Excel, SQL
- 能力词：财务建模, 估值分析, 行业研究, 尽职调查, 风险评估
- 成果词：投资回报率, 资产规模, 项目数量, 准确率

### 风控
- 工具：SAS, Python, SQL, 规则引擎
- 能力词：风险建模, 反欺诈, 信用评估, 合规审查
- 成果词：坏账率, 欺诈识别率, 审批效率
